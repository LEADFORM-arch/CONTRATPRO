# Frustrations chauffagistes observees sur les reseaux sociaux

Source : PDF fourni le 24/05/2026, synthese de recherches forums, Facebook,
YouTube, Trustpilot et avis en ligne autour des contrats d'entretien chaudiere.

Objectif : ne pas construire ContratPro autour d'une intuition. Chaque ecran
important doit repondre a une pression reelle du quotidien CVC.

## 1. Rush saisonnier et pannes d'automne

Frustration observee :

- vague d'appels au premier froid ;
- promesses d'intervention rapide difficiles a tenir ;
- planning sature, stress client et stress dirigeant.

Traduction ContratPro :

- montrer les contrats a reprendre avant le rush ;
- donner une file courte du jour ;
- eviter les tableaux trop larges quand l'artisan doit partir vite.

Message produit :

```text
Preparons les contrats avant que le froid ne transforme tout en urgence.
```

## 2. Rendez-vous, reports et clients absents

Frustration observee :

- confirmation, annulation, report ;
- client absent au domicile ;
- tournee desorganisee par un seul rendez-vous rate.

Traduction ContratPro :

- rattacher chaque contrat a une prochaine action claire ;
- conserver historique, contact, telephone, adresse et equipement au meme endroit ;
- faire ressortir les dossiers incomplets avant la visite.

Message produit :

```text
Le dossier doit dire qui appeler, ou aller, quoi verifier.
```

## 3. Devis complementaire et pression prix

Frustration observee :

- le client confond entretien et reparation ;
- le technicien passe pour "le mechant" quand une piece est a changer ;
- concurrence sur 15 ou 20 EUR sans perception de la valeur.

Traduction ContratPro :

- separer entretien, document, paiement et eventuel devis ;
- garder une trace claire de ce qui est inclus ou non ;
- aider le chauffagiste a justifier son suivi sans discours technique.

Message produit :

```text
Contrat d'entretien ne veut pas dire toutes reparations incluses.
```

## 4. Litige "avant ca marchait"

Frustration observee :

- panne apres intervention attribuee au technicien ;
- responsabilites floues entre entretien, depannage, locataire et proprietaire ;
- besoin de preuves simples et datees.

Traduction ContratPro :

- attestation, intervention, facture et paiement doivent rester lies au contrat ;
- chaque dossier doit servir de preuve de suivi ;
- les libelles doivent eviter les zones grises.

Message produit :

```text
Quand il y a contestation, le dossier doit parler pour vous.
```

## 5. Attestation sous 15 jours

Frustration observee :

- attestation obligatoire a fournir rapidement ;
- retard ou oubli pendant les periodes chargees ;
- reclamations assurance/garantie si le document manque.

Traduction ContratPro :

- faire apparaitre les attestations a envoyer comme une urgence documentaire ;
- creer un chemin direct contrat -> intervention -> attestation -> PDF ;
- ne pas cacher les documents dans un menu secondaire.

Message produit :

```text
Une visite terminee doit produire une preuve propre, pas une tache oubliee.
```

## Priorites produit issues de cette recherche

1. File du jour : une seule action prioritaire visible.
2. Dossier contrat : Facture, SEPA, Attestation en actions principales.
3. Documents : attestation sous 15 jours visible et rassurante.
4. Relances : langage "revenu a proteger" plutot que CRM generique.
5. Import : continuer a rassurer sur le dry-run avant creation.
6. Terrain mobile : garder le cap hors-ligne/photos/signature comme future etape,
   mais ne pas le bloquer avant le pilote.
7. Planning urgence : a garder pour plus tard, apres validation du coeur contrats.

## A ne pas faire maintenant

- construire un planning complet avant validation terrain ;
- promettre le hors-ligne complet trop tot ;
- ajouter de l'IA diagnostic panne avant d'avoir prouve la valeur contrats ;
- faire une usine a menus pour couvrir tous les cas.

## Phrase de positionnement mise a jour

```text
ContratPro aide les chauffagistes a retrouver le bon contrat au bon moment :
avant le rush, avant l'oubli, avant le litige, avant l'impaye.
```
