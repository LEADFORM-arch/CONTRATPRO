# Mode d'emploi operationnel

## Objectif

Utiliser Facebook comme canal fondateur pour obtenir les premiers rendez-vous
chauffagistes. Le skill aide a ecrire, prioriser et mesurer. Il ne remplace pas
la decision commerciale.

## Profil cible

Chercher des chauffagistes ou petites entreprises CVC qui ont :

- 50 a 300 clients ;
- des contrats d'entretien recurrents ;
- Excel, agenda, papier ou logiciel generaliste ;
- une douleur visible sur relances, attestations, impayes ou oublis ;
- un dirigeant joignable directement.

## Routine quotidienne

### Matin, 20 minutes

1. Ouvrir `/admin/prospection`.
2. Lire la file "Appels demo a traiter aujourd'hui".
3. Copier le DM froid depuis le panneau du skill.
4. Personnaliser le prenom, la ville et la douleur observee.
5. Envoyer 5 a 10 DMs maximum.
6. Ajouter les reponses ou prospects interessants dans `/prospection`.

### Midi, 15 minutes

1. Repondre aux commentaires sous les posts.
2. Qualifier les likes ou commentaires utiles.
3. Envoyer un DM seulement si le signal est clair.
4. Noter l'attribution : groupe, commentaire, like, demo publique.

### Soir, 20 minutes

1. Relancer les conversations ouvertes.
2. Passer les bons leads en `REPLIED` ou `DEMO_SCHEDULED`.
3. Copier dans les notes l'objection principale.
4. Preparer le post ou l'angle du lendemain.

## Cadence hebdomadaire

| Jour | Action | Sortie attendue |
| --- | --- | --- |
| Lundi 6h30 | Post probleme -> solution | Douleur concrete et lien demo trace |
| Mercredi 12h15 | Question d'engagement | Commentaires et signaux faibles |
| Vendredi 19h00 | Preuve, chiffre ou temoignage | Clics demo et DMs chauds |
| Chaque jour | 5 a 10 DMs cibles | Reponses et demos |

## Comment utiliser le cockpit

1. Panneau "Architecte IA prospection Facebook" : lire le module actif.
2. Copier DM ou post.
3. Adapter a une preuve observee : commentaire, ville, specialite, Excel.
4. Publier ou envoyer.
5. Reporter le resultat dans `/prospection`.
6. Revenir chaque semaine a la scorecard KPI.

## Ce qu'il ne faut pas faire

- Envoyer des DMs en masse sans personnalisation.
- Mettre un lien dans le premier DM froid.
- Parler de "notre solution" ou "notre plateforme".
- Publier plus de contenu que ce que tu peux suivre.
- Confondre likes et intention d'achat.
- Ajouter une feature produit pour une objection isolee.

## Decision apres 7 jours

Garder l'angle si :

- le taux de reponse DM depasse 15% ;
- au moins 2 conversations qualifiees existent ;
- une demo est planifiee ou proche.

Iterer si :

- les reponses existent mais l'objection revient ;
- les clics demo sont faibles ;
- le message attire des profils hors cible.

Couper si :

- aucun signal apres 50 DMs personnalises ;
- les commentaires parlent d'un besoin hors contrats ;
- le prix accepte reste sous 49 EUR/mois.
