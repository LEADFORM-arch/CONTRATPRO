# Skill prospection Facebook ContratPro

Ce dossier est le mode d'emploi du skill admin de prospection Facebook.
Il sert au fondateur pour trouver, qualifier et convertir les premiers
chauffagistes CVC, sans attendre Resend ni GoCardless.

Point d'entree dans l'application :

```text
/admin/prospection
```

Livrable Facebook complet :

```text
/facebook/contratpro-strategie-facebook.html
```

Ce fichier archive le kit Claude IA : identite de page, bandeaux, posts,
miniatures, calendrier, scripts Messenger, critique directeur de creation et V2
corrigee.

Le cockpit affiche le panneau "Architecte IA prospection Facebook" avec :

- les modules du skill : page Facebook, posts, DMs, KPI 90 jours ;
- un DM froid pret a copier ;
- un post probleme pret a publier ;
- une cadence hebdomadaire ;
- les seuils KPI pour garder, iterer ou couper une campagne.

## Ordre d'utilisation

1. Lire `mode-emploi.md` pour comprendre la routine quotidienne.
2. Utiliser `templates-copier-coller.md` pour publier ou envoyer des messages.
3. Utiliser `prompts-codex.md` pour demander a Codex de produire une variante.
4. Ouvrir `/facebook/contratpro-strategie-facebook.html` pour recuperer les
   textes V2, les prompts visuels et la checklist de lancement.
5. Remplir `kpi-scorecard.md` chaque semaine.
6. Reporter les leads chauds dans `/prospection`.

## Regle de pilotage

Ce skill ne doit pas devenir une usine a contenu. Il doit produire des preuves :

- messages envoyes ;
- reponses obtenues ;
- demos planifiees ;
- objections notees ;
- decision ecrite : garder, iterer ou couper.

Si une action ne cree pas un signal commercial mesurable, elle sort du skill.
