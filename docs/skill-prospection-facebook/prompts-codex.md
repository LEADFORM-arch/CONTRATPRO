# Prompts Codex pour utiliser le skill

Utiliser ces prompts dans Codex quand il faut produire une variante ou preparer
une campagne. Copier le contexte reel du prospect quand il existe.

## Generer un DM froid

```text
Utilise le skill prospection Facebook ContratPro.
Genere un DM froid de 4 phrases maximum pour ce prospect :

Prenom :
Ville :
Specialite :
Signal observe :
Douleur probable :

Contraintes :
- pas de lien ;
- ton pair-a-pair ;
- une seule question finale ;
- ne pas dire "notre solution".
```

## Generer un DM apres like

```text
Utilise le skill prospection Facebook ContratPro.
Genere un DM pour un chauffagiste qui a like un post sur les contrats oublies.

Contexte prospect :
[COLLER ICI]

Objectif : obtenir une reponse, pas vendre.
Sortie : message pret a envoyer, rien d'autre.
```

```text
[CORPS]
[CTA]
[HASHTAGS]
```

## Construire une semaine editoriale

```text
Utilise le skill prospection Facebook ContratPro.
Prepare une semaine de prospection Facebook avec 3 posts et une routine DM.

Contraintes :
- lundi 6h30, mercredi 12h15, vendredi 19h00 ;
- cible chauffagistes CVC France ;
- un post sans CTA produit pour engagement ;
- un post avec URL demo tracee ;
- indiquer le KPI de chaque action.
```
