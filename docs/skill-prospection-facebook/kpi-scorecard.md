## Lecture Architecte IA

### Vendre

Decision si :

- au moins 2 demos sont planifiees ;
- les prospects parlent bien de contrats, relances ou Excel ;
- le prix Starter a 49 EUR/mois n'est pas rejete immediatement ;
- une objection principale revient clairement.

Action : pousser les meilleurs leads dans `/prospection`, planifier les demos,
et utiliser `/admin/pilots` pour cadrer chaque rendez-vous.

### Iterer

Decision si :

- les reponses existent mais les demos ne sortent pas ;
- l'accroche attire des chauffagistes hors cible ;
- les posts generent des likes mais peu de conversations ;
- l'objection dominante est comprehensible.

Action : changer un seul parametre pendant 7 jours : accroche, cible, canal,
CTA ou angle.

### Couper

Decision si :

- moins de 10% de reponses apres 50 DMs personnalises ;
- les prospects veulent surtout une GMAO terrain complete ;
- le sujet contrats d'entretien ne les concerne pas ;
- le prix acceptable reste sous 49 EUR/mois.

Action : couper l'angle et choisir un nouveau segment ou un nouveau declencheur.
